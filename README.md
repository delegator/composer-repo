# Delegator - Composer Repository

Powered by [Satis][satis].

# Getting Started

```bash
# Install dependencies
$ make deps

# Compile repository
$ make build

# Preview the site at http://localhost:8080/
$ make serve

# Describe available make targets
$ make help
```
